## 主な特徴

### 初級プロンプト（🔰マーク付き）
- 専門用語を避け、わかりやすい言葉で説明
- 視覚的なゲージや絵文字を活用
- 「買い時/売り時」の断定を避け、中立的な表現
- 段階的な理解を促す構成

### 中級プロンプト
- 専門用語を適切に使用
- 数値と根拠を明確に提示
- 複数ツールを組み合わせた総合分析
- タイムスタンプを明記してデータの鮮度を担保

## 使い方

Claude Desktopなどのクライアントから、プロンプト名をそのまま呼び出せます：

```
🔰 最近のBTCの動きどう？
```

または、自然な質問でも適切なプロンプトが選択されます：

```
ビットコインの今後の見通しを教えて
```
